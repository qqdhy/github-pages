## MacBook DNS Config
Add public DNS servers in /etc/resolve.conf
```
nameserver 8.8.8.8
nameserver 180.76.76.76
nameserver 223.6.6.6
```
Flush DNS cache
```
sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder
```
